Un poco de historia

¿Qué es?

ASP.NET Core es un marco multiplataforma de código abierto y de alto rendimiento que tiene como finalidad compilar aplicaciones modernas conectadas a Internet y habilitadas para la nube. ASP.NET Core está desarrollado por Microsoft y la comunidad. Es un framework modular que se ejecuta completo tanto en el.NET Framework de Windows como en multiplataforma.

Lanzamiento

ASP.NET Core 1.0 finalmente se lanzó a fines de junio de 2016. Tomó más de dos años de desarrollo, pero el resultado fue ciertamente prometedor. Como diría Microsoft en su anuncio:

We challenged everything instead of delivering an incremental update so you can have an extremely modular, fast and lightweight platform perfect for the new era of software development where monolithic applications are replaced by small, autonomous services that can be deployed individually. All of that while keeping and extending what .NET is best for: developer productivity, and modern languages and libraries. […] The end result is an ASP.NET that you’ll feel very familiar with, and which is also now even more tuned for modern web development.

¿Por qué elegir ASP.NET Core?

Millones de desarrolladores usan o han usado ASP.NET 4 o superior para crear aplicaciones web. ASP.NET Core es un nuevo diseño del anteriormente mencionado que incluye cambios en la arquitectura que dan como resultado un marco más sencillo y modular. Ofrece, entre otras, las siguientes ventajas:

  • Un caso unificado para crear API web y una interfaz de usuario web
  • Diseñado para la capacidad de prueba
  • Capacidad para desarrollarse y ejecutarse en Windows, macOS y Linux
  • Herramientas que simplifican el desarrollo web moderno
  • Control de versiones en paralelo
  • Canalización de solicitudes HTTP ligera
