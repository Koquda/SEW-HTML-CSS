Ruta de aprendizaje recomendada

La ruta de aprendizaje recomendada por Microsoft es la siguiente:

  1. Hacer un nuevo desarrollo de interfaz de usuario web del lado servidor con Razor Pages
  2. Mantenimiento de una aplicación de MVC
  3. Desarrollar una interfaz de usuario del lado cliente con Blazor
  4. Crear una API web usando servicios HTTP RESTful
  5. Crear una aplicación de llamada a procedimiento remoto usando servicios de contrato primero con búferes de protocolo
  6. Crear una aplicación en tiempo real con comunicación bidireccional entre servidores y clientes conectados con SignalR
