Comparativa entre ASP.NET y ASP.NET Core

ASP.NET

ASP.NET 4.x es un marco consolidado que proporciona los servicios necesarios para compilar aplicaciones web de nivel empresarial basadas en servidor en Windows.

ASP.NET Core

ASP.NET Core es un marco multiplataforma de código abierto que tiene como finalidad compilar modernas aplicaciones web basadas en la nube en Windows, macOS y Linux.

Comparativa ASP.NET y ASP.NET Core
fila/columna ASP.NET ASP.NET Core
Compilación en Windows Windows, macOS, Linux
Cantidad de versiones 1 por equipo Varias por equipo
Entorno de desarrollo Visual Studio VS, VS para Mac y VSCode
Interfaz de usuario Web Forms, SIgnalR, MVC, API web, WebHooks o Páginas web Razor Pages (recomendado), MVC, Web API y SignalR
